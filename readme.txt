== Description ==
Plugin para registrar a personas interesadas en los servicios ofrecidos en la pagina web e ingresarlos directamente en la base de datos de Totalpat.com con el fin de realizar email-marketing y darle seguimiento en nuestro sistema de CRM, encuestas, Community Manager, etc.

== Installation ==
- Upload the entire Totalpat folder to the /wp-content/plugins/ directory.
- Activate the plugin through the \'Plugins\' menu in WordPress.
You will find \'Totalpat\' menu in your WordPress admin panel.
